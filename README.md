# 企业官网项目

## 项目简介
这是一个现代化的企业官网项目，采用响应式设计，提供专业的企业展示和客户交互功能。

## 技术栈
- HTML5 - 语义化标签和现代HTML结构
- CSS3 - 响应式设计、Flexbox、Grid布局、渐变效果
- JavaScript - 交互功能、平滑滚动、动态效果
- Font Awesome - 图标库
- Google Fonts - 字体优化

## 项目结构
```
test/
├── index.html          # 主页面
├── products.html       # 产品介绍页面
├── css/
│   ├── style.css       # 主样式文件
│   └── products.css    # 产品页面样式
├── js/
│   ├── main.js         # 主要JavaScript功能
│   └── products.js     # 产品页面功能
└── README.md           # 项目说明
```

## 功能特性
- 响应式导航栏
- 平滑滚动效果
- 现代化UI设计
- 移动端适配
- 交互式服务展示
- 联系信息展示
- 产品分类展示
- 产品详情模态框
- 产品对比表格
- 客户案例展示
- 在线咨询功能
